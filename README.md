# ShopSnap

ShopSnap is a web-based project that generates simple websites for local shops based on user input and shop category.

## Features
- Allows users to enter or select a shop (manually or via map concept)
- Generates a basic website layout based on shop type:
  - Grocery
  - Hardware
  - Food
  - Bakery, etc.
- Displays key shop information such as:
  - Opening and closing time
  - Popular items or dishes
  - Price range
  - Location details
  - Customer visited
- Includes user feedback/review concept
- Provides a structured website interface for small businesses

## Tech Used
- HTML
- CSS
- JavaScript

## About
This project was developed to explore how small businesses can quickly get a digital presence. It focuses on generating a simple website structure dynamically based on user input and shop category.
